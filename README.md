# Apple Pie Stamina Calculator

A simple static web calculator for FarmRPG players to compare eating Apple Pies now vs waiting for higher Tower/Exploring levels.

## Usage
- Open `index.html` in your browser.
- Or host the folder with GitHub Pages or Netlify for easy sharing.

## Files
- `index.html` — Main app
- `style.css` — Styles
- `script.js` — Calculator logic
- `applepie.png` — Icon

All calculations run locally in your browser.
